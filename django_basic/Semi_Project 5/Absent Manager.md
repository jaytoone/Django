## Absent Manager

### 출력 리스트

* 실시간 출석률

* 출결 석차

* 퇴소까지 남은 결석 시간, 몇번 빠질 수 있는지

* 일별 출석 시간, 특이사항 조회 (데이터 베이스에서 데이터 끌어오기)

* 일별 출석률

* 수료까지 남은 시간

  



### 코드 순서

1. 이름을 입력받으면 현재 주차까지의 데이터를 받아와서 작업한다.
2. 이름을 html 에서 받아와서 html 상에서 함수를 사용해 결과값을 출력한다.
3. 출력 리스트 출력
